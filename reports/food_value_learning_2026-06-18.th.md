## Experiment A — value-blind ingestion baseline

**design:** เพิ่ม food kind `raw_seed` (กินได้, พลังงาน ~1 vs ผล ~5, ขนาดเล็กพอดีทุกปาก), spawn 6/tick, v2, body 37, วัด diet ต่อ window โดยหาร availability ออก (`seed_frac` = สัดส่วนเมล็ดที่มีให้ ที่ถูกกิน)

| window | seed กิน | seed_frac | plant กิน | plant_frac | seed_share |
| --- | ---: | ---: | ---: | ---: | ---: |
| 0–1000 | 513 | 0.092 | 93 | 0.274 | 0.847 |
| 1000–2000 | 507 | 0.091 | 141 | 0.691 | 0.782 |
| 2000–3000 | 351 | 0.063 | 322 | 0.465 | 0.522 |
| 3000–4500 | 409 | 0.050 | 1114 | 0.471 | 0.269 |
| 4500–6000 | 375 | 0.046 | 1699 | 0.361 | 0.181 |

**ผล: กินแบบ value-blind — ไม่มีการเรียนรู้ว่าไม่คุ้ม**
- จำนวนเมล็ดค่าต่ำที่กิน **คงที่ ~375–510/window ตลอด 6000 ticks ไม่เคยดิ่งสู่ 0** (ถ้าเรียนรู้ จะต้องดิ่ง) — หลักฐานชี้ขาดว่าไม่มี learned avoidance
- มี selectivity (กินสัดส่วนผล > เมล็ด 3–7 เท่า) **ตั้งแต่ window แรก** = มาจาก sensory gradient (กลิ่นพลังงาน) ไม่ใช่การเรียนรู้ (ไม่ได้ค่อยๆ แรงขึ้น)
- seed_share ที่ลด 85%→18% อธิบายด้วย availability ของผลค่าสูงที่พุ่ง (กินผล 93→1699) ไม่ใช่การเลี่ยงเมล็ด

**confound ที่ต้องระวัง:** hunger ตัน (mean 0.983, 43–49/50 ตัวอยู่สถานะ hunger) → agent หิวตลอด นี่คือเหตุผลที่ Phase B ต้องแยกการตัดสินใจ "คุ้ม/ไม่คุ้ม" ออกจาก hunger flag

## Experiment B — เพิ่มกลไกเรียนรู้ค่าอาหาร (commit `d627bd4`)

**กลไก (จากประสบการณ์ ไม่ฮาร์ดโค้ด):** agent จำพลังงานที่เคยได้ต่อ food kind (EMA: `food_value_memory`) แล้วใช้ optimal-diet rule: กิน kind ที่ยังไม่รู้ค่า (ชิมก่อน) เสมอ; หลังรู้ค่าแล้ว ข้ามอาหารที่ค่า < `diet_pickiness` × ค่าที่ดีที่สุดที่รู้จัก มี starvation floor (กินทุกอย่างถ้าพลังงานต่ำกว่าเกณฑ์) — แยกจาก hunger flag ที่ตัน

### ผล 1: agent เรียนรู้ค่าถูกต้อง
`learned_food_value`: **seed = 1.0, plant = 5.0** ทุกรัน → จำได้ว่าเมล็ดแย่กว่าผล 5 เท่า

### ผล 2: learning curve (ปลด starvation override) — **เรียนรู้ที่จะเลิกกินจริง**

| window | seed กิน (B, override off) | seed กิน (A, ไม่มีกลไก) |
| --- | ---: | ---: |
| 0–500 | 125 | ~256 |
| 500–1000 | 103 | ~256 |
| 1000–2000 | 127 | 507 |
| 2000–3000 | **5** | 351 |
| 3000–6000 | **0** | ~1170 |

→ acquisition curve แบบตำรา: ชิมช่วงแรก → พอเรียนรู้ → **ดิ่งสู่ 0** (เลิกกินเมล็ดสนิทหลัง ~3000 ticks) หันไปกินผลค่าสูงเต็มที่ (กินผล 47→3209) **นี่คือคำตอบ: ถ้ามีกลไก มันเรียนรู้ว่า "ไม่คุ้ม" แล้วเลี่ยงได้จริง**

### ผล 3: แต่ภายใต้พลังงานจริง การเรียนรู้ถูกกลบ (key finding)
ถ้าใช้ starvation floor ปกติ (≥0): seed กิน/window = 498/463/425/467/436 — **เท่ากับ A เป๊ะ ไม่มี avoidance** เพราะการเทียบ floor=0 vs floor=−∞ ต่างกันมาก พิสูจน์ว่า **พลังงาน agent ≤ 0 แทบตลอดเวลา** (perma-starvation) → "eat anything when starving" เด้งทุกครั้ง → ต่อให้รู้ว่าเมล็ดแย่ ก็ยังกินเพราะกำลังจะอดตาย (เชิงเหตุผล)

## ข้อสรุปรวม (ตอบคำถาม)

1. **out-of-the-box: ไม่เรียนรู้** — การกินตัดสินแค่ขนาดปาก (value-blind), memory เชิงตำแหน่งเท่านั้น (A พิสูจน์: กินเมล็ดค่าต่ำคงที่ตลอดชีวิต)
2. **มีกลไก experience-driven value-learning: เรียนรู้ได้จริง** — เลิกกินเมล็ด (curve ดิ่งสู่ 0) โดยเรียนจากพลังงานที่ได้จริง ไม่ใช่ฮาร์ดโค้ด (B พิสูจน์)
3. **แต่ bottleneck คือพลังงาน** — ภายใต้เศรษฐกิจพลังงานปัจจุบัน agent อยู่ที่พลังงาน ≤ 0 ตลอด (perma-starvation) จึง "เลือกกินไม่ได้" (อดตายอยู่แล้ว) นี่คือ **รากเดียวกับที่บล็อก Phase 5 (site-selection), births=0, Phase 6** — แก้เศรษฐกิจพลังงานคือกุญแจปลดล็อกพฤติกรรมระดับสูงทั้งหมด

**Confidence:** กลไกเรียนรู้ทำงาน = สูง (curve ชัด, learned value ถูก) · พลังงานเป็น bottleneck = สูง (floor sweep พิสูจน์) · พร้อมใช้ในโลกจริง = ต่ำ (ต้องแก้พลังงานก่อน)
