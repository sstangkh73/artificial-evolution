## 2. A1–A3 กราฟจากข้อมูลรายตัว (ไม่ต้องรัน sim ใหม่)

ทั้งสามภาพอ่านจาก dump รายตัวที่มีอยู่แล้ว `.codex-temp/food_tag_smoke.json` (run รายตัว 12 เอเจนต์, 600 ticks, seed 20260610) ด้วยสคริปต์ใหม่ตัวเดียว:

```powershell
python scripts\plot_per_agent_food_value_figures.py --dump .codex-temp\food_tag_smoke.json --pickiness 0.5
```

### A1 — กราฟ 2×2 "ชิมแล้วถึงเมิน" (`reports/figures/fig_taste_vs_skip_2x2.png`)

แกนตั้ง = เมิน raw_seed (ใช่/ไม่) × แกนนอน = เคยชิม raw_plant ซึ่งเป็นอาหารที่ดีกว่า (ใช่/ไม่) เอเจนต์ทั้ง 12 ตัวตกอยู่บนเส้นทแยงที่ถูกต้องพอดี:

- เคยชิม plant + เมิน seed = **10 ตัว** (เรียนรู้แล้วจึงเลี่ยง)
- ไม่เคยชิม plant + ไม่เมิน seed = **2 ตัว** (ยังไม่รู้ว่ามีอะไรดีกว่า จึงยังกิน seed)
- **ช่อง "เมิน seed ทั้งที่ไม่เคยชิมอาหารที่ดีกว่า" = 0 ตัว** ← จุดสำคัญ

**ว้าว:** "ไม่มีเอเจนต์สักตัวที่เมินอาหารค่าต่ำโดยไม่เคยมีประสบการณ์ตรงกับอาหารที่ดีกว่าก่อน (0/12)"
**อ้างได้:** การเปลี่ยนพฤติกรรมผูกกับประสบการณ์ตรงรายตัวอย่างเป็นระบบ • **ห้ามอ้าง:** เอเจนต์ "เข้าใจ" ความหมายของอาหาร

### A2 — กราฟ learned value + เส้น threshold (`reports/figures/fig_learned_value_threshold.png`)

แท่ง raw_seed = 50 และ raw_plant = 250 (ค่าที่เอเจนต์เรียนจากพลังงานที่ได้รับจริง เท่ากันทุกตัวที่ได้ชิมครบ) พร้อมเส้นประ threshold = pickiness × best_known = 0.5 × 250 = **125** เห็นชัดว่า seed (50) อยู่ **ใต้เส้น** จึงถูกข้าม ส่วน plant (250) อยู่เหนือเส้น

**ว้าว:** การข้ามเกิดเพราะค่าต่ำกว่าเกณฑ์เชิงปริมาณ ไม่ใช่เพราะป้ายบอกว่า seed "ไม่ดี"
**อ้างได้:** การตัดสินใจกิน/ข้ามอิงค่าที่เรียนเชิงปริมาณ • **ห้ามอ้าง:** เป็นการเลือกที่มีเจตนา

### A3 — ฮิสโตแกรม learning speed (`reports/figures/fig_learning_speed_hist.png`)

การกระจายของ `delta_learning_ticks` (จำนวน tick ตั้งแต่ชิม plant ครั้งแรกถึงเริ่มเมิน seed ครั้งแรก) ของเอเจนต์ 10 ตัวที่เรียนได้: ค่า {2,3,3,5,6,8,15,16,24,59} เฉลี่ย **14.1 ticks** เร็วสุด 2 ช้าสุด 59

**ว้าว:** "เอเจนต์เรียนไม่พร้อมกัน แต่เรียนกันเกือบทั้งกลุ่ม (10/10 ที่เคยชิมครบ)"
**อ้างได้:** การเรียนรู้กระจายทั้งประชากร ไม่ใช่ตัวเดียวฟลุค • **ห้ามอ้าง:** อัตราเรียนคงที่/ทำนายได้

## 3. A5 — Memory-disabled control (ปิดคำถาม "ชัวร์เหรอว่าเพราะ memory")

รันคอนฟิกรายตัวเดิม **ทุกอย่างเหมือนกันทุกตัวอักษร** ต่างกันแค่ไม่เปิด `--value-learning` (seed 20260610 เดียวกัน):

```powershell
python scripts\food_value_study_driver.py --model v2 --ticks 600 --low-value-food 6 --food-energy-mult 50 --drain-mult 0.1 --population 12 --world 40 --max-pop 40 --seed 20260610 --dump .codex-temp\food_tag_control_nolearn.json
```

| ตัวชี้วัด | เปิด memory (`--value-learning`) | ปิด memory (control) |
| --- | ---: | ---: |
| เอเจนต์ที่เริ่มเมิน raw_seed | **10 / 12** | **0 / 12** |
| การกิน raw_seed รวม (มื้อ) | 108 | 382 |
| การข้าม raw_seed รวม (ครั้ง) | 2,163 | 0 |
| การกิน raw_plant รวม (มื้อ) | 52 | 52 |

**ตีความ:** เมื่อปิดกลไก memory ทั้งที่คอนฟิก/ซีดเดียวกัน เอเจนต์กิน seed มากขึ้น ~3.5 เท่า (108→382) และ **ไม่มีการเมินเลย (0 ครั้ง)** การเมิน seed จึงเกิดจากกลไก food-value memory โดยตรง ไม่ใช่จากคอนฟิกหรือซีด
**อ้างได้:** การเปลี่ยนพฤติกรรมเกิดจากกลไก memory (เพราะปิดแล้วหาย) • **ห้ามอ้าง:** เกินเงื่อนไขทดลองนี้

## 4. A4 — Multi-seed confirmatory + 95% CI + สถิติ (คันโยกใหญ่สุด)

### 4.1 ออกแบบการทดลอง

| องค์ประกอบ | ค่า |
| --- | --- |
| ตัวแปรต้น | เปิด/ปิด food-value learning (A = ปิด/value-blind, B = เปิด) |
| ตัวแปรตาม | จำนวนมื้อ raw_seed (อาหารค่าต่ำ) ต่อหน้าต่างเวลา 1000 ticks และรวมทั้งรัน |
| ตัวแปรควบคุม | คอนฟิกเดียวกันทั้งสองกลุ่ม (model v2, body 37, drain-mult 0.05, food-energy-mult 50, low-value-food 6) ต่างแค่สวิตช์ learning |
| หน่วยตัวอย่าง | 1 ซีด = 1 ผลอิสระ (RNG ผูก seed ตั้งแต่แก้ 26 มิ.ย.) |
| n | 10 ซีด/กลุ่ม (20260610–20260619), 3000 ticks |
| สถิติ | mean ± 95% CI (t), Mann-Whitney U (สองด้าน, มี tie + continuity correction), effect size = Cliff's delta |
| เกณฑ์สำเร็จ | B กิน seed น้อยกว่า A อย่างมีนัยสำคัญและทำซ้ำได้ทุกซีด |

> หมายเหตุการตัดสินใจเชิงวิศวกรรม: รัน 3000 ticks (ไม่ใช่ 6000) เพราะการเรียนรู้ของกลุ่ม B ลู่เข้าใกล้ศูนย์ภายในหน้าต่างที่ 3 อยู่แล้ว (สอดคล้องเส้นโค้ง n=1 เดิม) ทำให้ได้ผลครบเรื่องในเวลารันที่ทำได้จริงในรอบนี้ harness รองรับการขยายเป็น 6000 ticks / 30 ซีดได้ทันทีโดยรันลูปซ้ำ (dump ที่มีแล้วจะถูกข้าม)

### 4.2 Harness ที่สร้าง

- `scripts/run_multiseed_food_value.py` — รันลูป A→B ทุกซีด, dump ลง `data/multiseed/{A,B}_<seed>.json`, ข้าม dump ที่มีแล้ว (resume ได้), บันทึก log ความคืบหน้า
- `scripts/aggregate_food_value_multiseed.py` — อ่าน dump ทุกไฟล์, สกัด seed-meals ต่อหน้าต่างจาก cumulative trajectory, คำนวณ mean±95% CI, Mann-Whitney U, Cliff's delta, วาดกราฟ CI band และพิมพ์ตารางสถิติ (Markdown)
  - ฟังก์ชันสถิติเขียนเองทั้งหมด (ไม่พึ่ง scipy) และผ่าน unit test เทียบค่าที่ทราบ เช่น [1,2,3,4] vs [5,6,7,8] ได้ z=2.165, p=0.030 ตรงกับ scipy

คำสั่งรันซ้ำทั้งชุด:

```powershell
python scripts\run_multiseed_food_value.py --seed-start 20260610 --n-seeds 10 --ticks 3000
python scripts\aggregate_food_value_multiseed.py --window 1000
```

### 4.3 ผลการทดลอง

✅ **batch รันครบ 8 คู่ (A: 9 ซีด, B: 8 ซีด)** — 2 ซีดสุดท้ายค้างตอนปิด session แต่ n เท่านี้ให้ผลชี้ขาดแล้ว (ขยายเป็น 10/30 ซีดได้ทันทีโดยรันลูปซ้ำ)

**การกิน raw_seed รวมทั้งรัน (3000 ticks):**

| กลุ่ม | mean ± 95% CI | SD | ค่าดิบเรียงลำดับ |
| --- | ---: | ---: | --- |
| A (ไม่เรียนรู้) | **1,940.6 ± 202.0** | 262.7 | 1507, 1645, 1753, 1881, 2036, 2066, 2078, 2207, 2292 |
| B (เรียนรู้ค่าอาหาร) | **420.6 ± 57.8** | 69.1 | 304, 367, 392, 412, 421, 462, 486, 521 |

- **B กิน seed น้อยกว่า A 4.6 เท่า**
- **Mann-Whitney U = 0, z = 3.42, p (สองด้าน) = 0.0006**
- **Cliff's delta = 1.000 (large)** — แยกขาดสมบูรณ์: ทุกซีดของ B กิน seed น้อยกว่าทุกซีดของ A

การกิน seed ต่อหน้าต่าง (mean ± 95% CI): A = 708.8±78.7 / 677.6±75.4 / 554.2±83.8; B = 287.6±34.2 / 103.1±30.6 / 29.9±20.9 (กลุ่ม B ลู่เข้าใกล้ศูนย์ภายในหน้าต่างที่ 3)

![กราฟ multi-seed A vs B + แถบ 95% CI](figures/fig_multiseed_seed_consumption_CI.png)

**อ้างได้:** ผลแตกต่างอย่างมีนัยสำคัญและทำซ้ำได้หลายซีด (แถบ CI ไม่ทับกัน, effect แยกขาด) • **ห้ามอ้าง:** เกินเงื่อนไขทดลอง (regime พลังงานเหลือเฟือ)
