# แผนเส้นทาง YSC → ISEF สำหรับโครงการ Artificial Evolution

**วันที่:** 2026-06-27
**ผู้จัดทำ:** Chisanupong (ม.4 โรงเรียนดีบุกพังงาวิทยายน)
**สถานะโครงการ:** มีผลหลัก (food-value learning + per-agent) + เพิ่ง implement neural network + neuroevolution (4.9× learning curve)
**เป้าหมาย:** ผ่าน YSC ระดับชาติ → ได้เป็นตัวแทนไทยไป Regeneron ISEF

> เอกสารนี้ตอบ 2 คำถาม: (1) **ไหวไหม** ตามจริง และ (2) **ทำอะไรบ้าง** ทีละขั้นจนถึง ISEF
> ภาษาในเอกสารตรงไปตรงมา — ตัวเลขโอกาสจริง ไม่กล่อม เพราะความซื่อสัตย์คือจุดแข็งของโครงการนี้

---

## 0. สรุปผู้บริหาร (อ่านก่อน)

**ไหวไหม?** — ISEF คือ **stretch goal ที่มีเส้นทางจริง** ไม่ใช่ฝันเพ้อ แต่ต้องเข้าใจสเกล:

- YSC ครั้งล่าสุดมี ~2,429 โครงงาน / 6,442 คน → คัดเป็นตัวแทนไทยไปเวทีนานาชาติ (ISEF + Genius Olympiad + Geneva รวมกัน) เพียง **~8 โครงงาน / 18 คน** = ราว **0.3% สุดยอด**
- แปลว่า "YSC → ISEF" = ต้องเป็น **หัวกะทิระดับประเทศ** ไม่ใช่แค่ "ผ่านเข้ารอบ"

**ความจริงที่ให้กำลังใจ:** โปรเจกต์คุณมี "DNA" ที่ ISEF มองหาจริง — original (ALife/emergence แทบไม่มีในสนามไทย), computational, มี control + ความซื่อสัตย์เชิงวิธีวิทยา และเป็น **งานคอมพิวเตอร์ล้วน** → **ไม่ติดเอกสารจริยธรรม/ความปลอดภัยชีวภาพ** ที่ถ่วงคู่แข่ง wet-lab จำนวนมาก

**ความจริงที่ต้องยอมรับ:** ตอนนี้โปรเจกต์อยู่ระดับ "ดีมากสำหรับโรงเรียน" ยังไม่ใช่ "แน่นระดับชาติ" — ช่องว่างหลักคือ **สถิติ/ความเข้มเชิงปริมาณ** และ **การนำเสนอภาษาอังกฤษ** ถ้าอุด 2 อย่างนี้ได้ โอกาสเปลี่ยนจาก "เป็นไปไม่ได้" เป็น "เป็นผู้ท้าชิงจริง"

**กลยุทธ์ความคิด:** ตั้งเป้า ISEF เป็น "ยอดเขา" แต่นิยามความสำเร็จเป็น **หลายระดับ** (ดู §8) — เพราะการอัปเกรดที่ต้องทำเพื่อ ISEF จะทำให้คุณเป็นผู้ชนะระดับภูมิภาค/ชาติได้ไม่ว่าจะถึง ISEF หรือไม่

**ไทม์ไลน์เป้าหมายจริง: ISEF 2027** (จัด ~พ.ค. 2027) ผ่าน **YSC ครั้งที่ 28 (รอบปี 2026→2027)** — ทันพอดีถ้าเริ่มเดี๋ยวนี้

## 2. การประเมินช่องว่าง (ตอนนี้ vs ระดับ ISEF)

| มิติ | ระดับปัจจุบัน | ระดับ ISEF ต้องการ | ช่องว่าง |
|---|---|---|---|
| **ความใหม่ของคำถาม** | สูง (ALife/unlabeled/evolution) | สูง + วางตำแหน่งเทียบงานสากล | 🟡 ต้องมี related work จริง |
| **สถิติ/ความเข้มเชิงปริมาณ** | **n=1 deterministic** | multi-seed + mean±SD + นัยสำคัญ + CI | 🔴 **ช่องว่างใหญ่สุด** |
| **Reproducibility** | ไม่ deterministic ถ้าไม่ล็อก PYTHONHASHSEED; `args.seed` ไม่ผูก movement RNG | reproducible + seed ทำงานจริง | 🔴 ต้องแก้ |
| **ผลที่จับต้อง/เห็นภาพ** | มีกราฟ + เดโม่ random vs evolved แล้ว | figure คุณภาพสูง + เดโม่ชัด | 🟢 เริ่มแล้ว ต่อยอดได้ |
| **เรื่องเล่า/contribution** | ผลกระจาย (food-value, neural แยกกัน) | เรื่องเดียวที่คม | 🟡 ใช้กรอบ "2 ระดับเวลา" |
| **ความซื่อสัตย์เชิงวิธีวิทยา** | **แข็งมาก** (claimable/not, falsification) | แข็ง | 🟢 จุดแข็ง รักษาไว้ |
| **การสื่อสาร (English)** | ส่วนใหญ่เป็นไทย | report/poster/พูด/ตอบ เป็นอังกฤษระดับมืออาชีพ | 🔴 ต้องลงทุน |
| **mentor/ที่ปรึกษา** | เดี่ยว ม.4 + ครูที่ปรึกษา | ที่ปรึกษาเชิงสถิติ/วิจัยที่เข้ม | 🟡 ควรหาเสริม |
| **เอกสาร/โลจิสติกส์** | ยังไม่เริ่มสาย ISEF | ฟอร์ม ISEF + สมัคร YSC ทัน | 🟡 เริ่มเดี๋ยวนี้ |

**บทสรุปช่องว่าง:** 2 เรื่องสีแดงที่ขวางระหว่าง "โรงเรียน" กับ "ชาติ/ISEF" คือ **สถิติ** และ **ภาษาอังกฤษ** — ที่เหลือคือต่อยอดจากของที่มี

## 3. แผนแก้เป็นเฟส (ทำอะไร เมื่อไหร่)

### Phase 0 — เก็บแต้มเวทีปัจจุบัน + ตั้งหลัก (ตอนนี้ → ก.ค. 2026)

เป้าหมาย: ส่งงานเวทีที่มี + วางรากฐานสาย ISEF

- [ ] **ส่งโครงงานวิทย์ครั้งที่ 5** (เส้นตาย 17 ก.ค. 2026): รายงานเต็ม + บทคัดย่อ + วิดีโอ ≤15 นาที
- [ ] **ส่ง/ตามผล TURC** (ประกาศ 29 มิ.ย.) — ไม่ว่าผลใดก็เดินต่อ
- [ ] **สมัคร YSC ครั้งที่ 28 ผ่าน SIMS ทันที** (ยืนยันเส้นตายในเว็บ NSTDA) ← **ห้ามพลาด นี่คือประตูสู่ ISEF**
- [ ] **ตัดสินใจ scope งานวิจัยให้อยู่ในหน้าต่าง ISEF** (ม.ค. 2026 เป็นต้นไป): ใช้ neural/neuroevolution/communication เป็นแกน
- [ ] Quick win: รัน multi-seed ของผลที่มี + ทำ figure มีแถบความเชื่อมั่น (เริ่มอุดสถิติ)

### Phase 1 — อุดสถิติ + reproducibility (ก.ค. → ก.ย. 2026) 🔴 KEYSTONE

เป้าหมาย: เปลี่ยนทุกผลจาก n=1 เป็นมีสถิติรองรับ

- [ ] **ผูก `args.seed` เข้า movement RNG** (`Random(agent_id+age)` → รวม seed) เพื่อให้หลาย seed เป็น replicate จริง
- [ ] **แก้ reproducibility**: ตั้ง/บันทึก `PYTHONHASHSEED`, ทำให้รันซ้ำได้ byte-identical, เขียน seed policy ใน README
- [ ] **รัน multi-seed (≥10–20 seeds)** สำหรับทุกผลหลัก:
  - food-value learning (A vs B)
  - neuroevolution (learning curve, random vs evolved)
- [ ] **คำนวณสถิติ**: mean ± SD, 95% CI, การทดสอบนัยสำคัญ (Mann-Whitney U เหมาะกับ n เล็ก/ไม่ปกติ; รายงาน effect size)
- [ ] **ทำ figure ใหม่**: ทุกกราฟมีแถบ CI/error bar
- [ ] เขียน "Statistics & Reproducibility" section ในรายงาน

**Definition of done:** ทุก claim หลักมีตัวเลข "n=..., p=..., CI=..." รองรับ

### Phase 2 — ความลึกทางวิทยาศาสตร์ + เรื่องเล่า (ก.ย. → ธ.ค. 2026)

เป้าหมาย: ทำให้ contribution คมและลึกพอระดับชาติ

- [ ] **กรอบเรื่องเดียว "การเรียนรู้ 2 ระดับเวลา"**: ภายในช่วงชีวิต (food-value) + ข้ามรุ่น (neuroevolution) — เล่าซื่อสัตย์ว่าเป็น 2 การศึกษาบนแพลตฟอร์มเดียว
- [ ] **สร้างการศึกษาการสื่อสาร (signal channel)** บน NN: เพิ่ม signal output + signal input → วิวัฒนาการ → ทดสอบด้วย control (signal-shuffle, receiver-blind) → ตอบ H2 ที่เคย "ทดสอบไม่ได้"
- [ ] **related work จริง**: Avida, Polyworld, NEAT, Lewis signaling games / emergent communication, optimal foraging — วางว่างานคุณต่าง/เสริมอย่างไร
- [ ] **rebalance พลังงานแบบมีหลักการ** (แทนคันโยกเทียม food×50/drain÷20) เพื่อ ecological validity
- [ ] (ถ้าทำได้) **รวมการเรียนรู้ภายในชีวิต + วิวัฒนาการในเอเจนต์ NN ตัวเดียว** → เรื่องเล่าสมบูรณ์ขึ้น

**Definition of done:** มีผลใหม่ ≥1 (การสื่อสาร) + related work + เรื่องเล่าที่ผูกทุกผลเป็นหนึ่ง

### Phase 3 — เตรียมแข่ง YSC + ภาษาอังกฤษ (ธ.ค. 2026 → มี.ค. 2027)

เป้าหมาย: แปลงงานวิจัยเป็นการนำเสนอที่ชนะระดับชาติ

- [ ] **รายงาน + abstract เป็นภาษาอังกฤษ** ตามฟอร์แมต ISEF (RQ/procedures/data/interpretation/conclusions)
- [ ] **โปสเตอร์ภาษาอังกฤษ** + เดโม่เห็นภาพ (learning curve, random vs evolved, communication)
- [ ] **ซ้อมพูด + ตอบ Q&A เป็นอังกฤษ** โดยเฉพาะ: อธิบาย NN/neuroevolution/สถิติ ให้ลึกจริง (กรรมการเจาะแน่)
- [ ] **หา mentor เสริม** ด้านสถิติ/ML/ภาษาอังกฤษ (อาจารย์มหาวิทยาลัย/นักวิจัย) — solo ม.4 ควรมีพี่เลี้ยงเชิงเทคนิค
- [ ] เข้าร่วมรอบ YSC ตามกำหนด (ข้อเสนอ → ภูมิภาค → ชิงชนะเลิศ)

**Definition of done:** นำเสนอ + ตอบ Q&A เป็นอังกฤษได้มั่นใจ; เข้ารอบลึกใน YSC

### Phase 4 — ค่ายคัดตัวแทน + ISEF (มี.ค. → พ.ค. 2027) *(ถ้าผ่านระดับชาติ)*

- [ ] เตรียมเอกสาร ISEF: Checklist Form 1, 1A, Research Plan, Approval 1B (งาน sim = ฟอร์มเบา)
- [ ] ปรับ display board + abstract ให้ "เฉพาะงานในหน้าต่าง 12 เดือน"
- [ ] ซ้อมนำเสนอกับกรรมการนานาชาติ (อังกฤษ, เชิงลึก, เปรียบเทียบงานสากล)

## 4. จุดแข็งที่ "ห้ามทำหาย" ระหว่างอัปเกรด

1. **ความซื่อสัตย์เชิงวิทยาศาสตร์** (ตาราง claimable/not, falsification, honest negative) — นี่คือเอกลักษณ์
2. **ความ original** (ALife/emergence ไม่ซ้ำใครในสนามไทย)
3. **ความเป็น stdlib/reproducible** (รันง่าย ตรวจสอบได้)
4. **เดโม่เห็นภาพ** (random vs evolved) — ทำให้ของนามธรรมจับต้องได้

## 6. นิยามความสำเร็จแบบหลายระดับ (อย่าวัดที่ ISEF อย่างเดียว)

ตั้ง ISEF เป็นยอดเขา แต่ "ชนะ" ได้หลายระดับ — ทุกระดับคือของจริงที่ภูมิใจได้:

- 🥉 **ระดับ 1:** ผ่านเข้ารอบ TURC / ได้เหรียญครั้งที่ 5 + โปรเจกต์มีสถิติรองรับ
- 🥈 **ระดับ 2:** เข้ารอบลึก YSC / ได้รางวัลภูมิภาค + มีผลการสื่อสาร + related work (= งานระดับชาติจริง)
- 🥇 **ระดับ 3:** ชนะสาขา YSC ระดับชาติ → เข้าค่ายคัดตัวแทน
- 🏔️ **ยอดเขา:** ได้เป็นตัวแทนไทยไป ISEF 2027

> แม้ไปไม่ถึงยอด การอัปเกรดทั้งหมดทำให้คุณมี "งานวิจัยจริง" ที่ใช้ต่อยอดได้ถึงมหาวิทยาลัย — นั่นคือชัยชนะที่ไม่มีใครเอาคืนได้

## 8. ลำดับความสำคัญถ้าทำได้แค่ 3 อย่าง
