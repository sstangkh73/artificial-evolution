state of the repository and the latest experiment outputs as of `2026-05-11`.

- `paper_summary_en.md`: English narrative draft

This package is intended to bridge:

- the achievements of Phase 1
- the post-Phase-1 transition into sexed reproduction and emergent technology
- the current unresolved problems
